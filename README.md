### 🛠 Инструкция по установке
> **Важно:** Установка полностью аналогична оригинальной версии утилиты.

1. Распакуйте архив в удобную папку.
2. Запустите файл `service.bat` от имени Администратора.
3. В появившемся меню выберите пункт **Install Service**.
4. Установите/запустите нужный вам `.bat`-файл из комплекта.
5. **Готово!** Проверяйте работу YouTube, Discord и Fortnite.
